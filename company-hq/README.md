## How This Directory Relates To The Repo Root

The full firm operating truth is split across:

- root strategic baton files such as `CURRENT.md`, `MEMORY.md`, `SOUL.md`, `TRINITY.md`
- root doctrine files in `doc/plans/`
- company-level governance and vision files here in `company-hq/`

Use `company-hq` for:

- compact firm heart and vision
- governance rules
- company-specific operating policies
- stable capability and research context

Use the repo root for:

- live baton
- stable cross-session memory
- direct assistant identity docks
- execution runbook and repo-local operating truth
