## Workspace Truth

- Package manager: `pnpm`
- Node: `>=20`
- TypeScript: strict
- Main app entry commands:
  - `pnpm install`
  - `pnpm dev`
  - `pnpm dev:once`
  - `pnpm build`
  - `pnpm typecheck`
  - `pnpm test:run`
- Local dev runtime:
  - API server: `http://localhost:3100`
  - UI: served from the same origin in dev middleware mode

## Stable Run Path

- For stable local verification of heartbeat/runtime behavior, prefer `pnpm dev:once` over `pnpm dev`.
- Use `pnpm dev` when normal watch mode is wanted.
- Do not assume watch mode is safe for long-running runtime verification.
